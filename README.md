# TaskManager

This project was paused during development.

It was being used as a learning project for Spring Boot, JPA, service-layer design, custom exceptions, and permission rules. As the scope grew, the codebase started to become harder to evolve safely because several parts were still incomplete or only partially tested.

Current status:

- the project is not considered finished
- the architecture is still in progress
- there are known inconsistencies and missing tests
- it should be treated as a study/prototype project, not a production-ready application

Why it was paused:

- the scope became larger than expected
- the project structure needs a cleaner foundation
- development was moving faster than testing and validation

Planned direction for the future:

- return after more practice with smaller Spring Boot projects
- rebuild or refactor parts of the application with better test coverage
- improve consistency across services, permissions, and API behavior

If you are looking at this repository, the best way to understand it is as an intermediate learning project rather than a completed system.
